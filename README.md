# ‍💼 «X-Zone»

> Landing page for a company aimed at the VR industry.

## 🖥️ Deployment

To deploy this project run

```
$ yarn
$ yarn run dev
```

## 🛠 Skills

- Pug.
- TailwindCSS.
- Sass (SCSS syntax).
- JavaScript.

## 🛠 Features

- Using WebP images.
- Using Flex.
- Using Tailwind CSS framework
- Local fonts.
